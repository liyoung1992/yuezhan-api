### 添加战队

URL：http://127.0.0.1:8080/v1/team/add-team

1.请求参数



```go



{

    token: "token",

    key: "key",

    teamName: "无敌战队",

    regionId: 11,

    userId: 12,

    leaderId: 121,

    createTime: 1486284225,

    imageUrl: "1212111111"

}



```



2.返回参数



```go



{

    Result: 0,

    Err: "ok"

}



```
